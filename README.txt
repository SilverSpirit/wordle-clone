# wordle-clone
Lazy Python wordle clone. Run with python wordle.py

[] denotes a letter in the right spot, () indicates a letter in the wrong spot.

Enter a valid 5 letter word:

WEIRS

__________________
(S)  T   O  [R] (E)
(R)  A   C  (E)  R
 P  [E]  A  [R] [S]
 V  [E]  E  [R] [S]
[W] [E] [I] [R] [S]
_ _ _ _ _
__________________

Win!
